# production-polish — plan

Each code step ships as one executor run (`/feature small`, `/feature`, or `/upgrade`)
in its own tab, producing one PR against `master`. The planner writes the step prompt,
reviews the executor's plan & design, then reviews the PR.

The **design track** runs in Claude Design from planner-written briefs
(`design-*-prompt.md`). D1 delivered the ratified system; per D-11 the old D2 kit
screens are demos, and D3 designs real pages one at a time — each page's
implementation step is gated on its D3 ratification.
